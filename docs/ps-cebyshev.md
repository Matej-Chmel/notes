# Čebyševova nerovnost

Na základě směrodatné odchylky $\sigma$ si lze vytvořit představu o rozložení pravděpodobnosti NV.

\[ \begin{align}
\forall k > 0 &= P(\mu - k\sigma < X < \mu + k\sigma) &\geq 1 - \frac{1}{k^2}
\end{align} \]

|$k$|$P(\mu - k\sigma < X < \mu + k\sigma)$|
|--:|--:|
|$1$|$\geq 0,00$|
|$2$|$\geq 0,75$|
|$3$|$\geq 0,89$|
